Descrição do Projeto

Este projeto consiste no desenvolvimento de um sistema web utilizando a linguagem PHP com arquitetura baseada no padrão Model-View-Controller,
proporcionando melhor organização do código, separação de responsabilidades e maior facilidade de manutenção e escalabilidade.

A aplicação utiliza PHP Data Objects para a conexão e manipulação segura do banco de dados, garantindo maior proteção contra vulnerabilidades como 
SQL Injection. No desenvolvimento da interface foi utilizado o framework Bootstrap, permitindo a criação de layouts modernos, responsivos e compatíveis 
com diferentes dispositivos.

O projeto tem como objetivo demonstrar boas práticas de desenvolvimento web, incluindo organização estrutural do sistema, integração com banco de dados, 
reutilização de componentes e construção de interfaces amigáveis ao usuário.
